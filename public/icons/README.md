# PWA Icons and Splash Screens

This folder contains the PWA icons for the RentCare app.

## Required Icons

For optimal PWA support, create the following icons from your logo:

### App Icons (in /public/icons/)
- `icon-192.png` - 192x192 pixels
- `icon-512.png` - 512x512 pixels

### Apple Touch Icon (in /public/)
- `apple-touch-icon.png` - 180x180 pixels

### Splash Screens (in /public/splash/)
For iPhone splash screens, create:
- `splash-1170x2532.png` - iPhone 12/13/14 (390x844 @3x)
- `splash-1284x2778.png` - iPhone 12/13/14 Pro Max (428x926 @3x)

## Current Setup
Currently using `logo.png` as a placeholder for all icons.
For production, replace these with properly sized versions.
